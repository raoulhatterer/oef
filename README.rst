oef-mode
==================================

provide oef-mode (Online Exercise Format for wims) to emacs  

Features (Non-exhaustive):
--------------------------

* Syntax higligtment 
* Automatic indentation
* Connction to a wims session
* OEF Menu with:
 * Examples
 * Usefull Characters shortcuts
 * OEF Commands and Keywords
